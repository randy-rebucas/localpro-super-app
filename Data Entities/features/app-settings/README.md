# App Settings Feature

## Overview

Global configuration for the platform: feature flags, security policies, business info, uploads, notifications, payments, analytics, and integrations.

## Key Capabilities
- Centralized admin-managed settings
- Public subset for client bootstrap and health
- Category-level updates and feature flag toggles
- Versioned and environment-aware metadata
