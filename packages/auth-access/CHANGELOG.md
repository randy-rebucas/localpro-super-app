# Changelog

All notable changes to @localpro/auth-access will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.1] - 2025-12-31

### Added
- Full TypeScript support with type definitions
- TypeScript definitions file (index.d.ts)
- Support for React.js and Next.js projects
- Type-safe Express request extensions
- Type-safe GraphQL context types
- TypeScript configuration file

## [1.0.0] - 2025-12-31

### Added
- Initial release of @localpro/auth-access
- JWT token management with RS256 (asymmetric signing)
- Token issuance, validation, and refresh flows
- Role-Based Access Control (RBAC) with scopes
- Express.js middleware support
- Fastify middleware support
- GraphQL/Apollo Server integration
- Utility functions for token and scope validation
- Support for custom scopes
- Role hierarchy system
- Comprehensive error handling

### Features
- **Token Management**: Issue, validate, and refresh JWT tokens
- **RBAC**: Role and scope-based access control
- **Middleware**: Express, Fastify, and GraphQL support
- **Security**: RS256 asymmetric key signing
