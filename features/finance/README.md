# Finance Feature

## Overview

User financial operations: loans (incl. salary advance), wallet and transactions, withdrawals, and financial analytics/reports. Supports multiple payment methods including PayMongo for secure payment processing.

## Modules
- Loans: salary advance, micro/business/equipment loans, lifecycle and repayments
- Salary Advance: short-term advance tied to payroll
- Transactions: system-level and wallet transaction records with PayMongo support
- Finance (Wallet): per-user balances, transactions, and settings
- Payment Processing: PayMongo integration for top-ups and withdrawals
